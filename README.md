# Trimly

Trimly is a Web3-styled beauty & grooming ecosystem app.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Animations:** Framer Motion
- **Utilities:** clsx, tailwind-merge

## Design Language

- **Mobile-First:** Optimized for mobile screens (max-width: 480px centered on desktop).
- **Theme:** Modern Web3 (Dark Mode base).
- **Visuals:** Glassmorphism, rounded corners, thin borders.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) (or the port shown in terminal) with your browser to see the result.
